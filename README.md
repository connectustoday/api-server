# ConnectUS Backend API Server

## Running (Development)
The server software has only been tested on GNU/Linux.

```bash
$ git clone https://github.com/connectustoday/api-server.git
$ cd api-server
$ ./build.sh
$ ./run.sh

```

## Running (Production)
It is recommended to use our docker image.
